# IPL Data Analysis Dashboard

## Project Overview
This project analyzes IPL (Indian Premier League) match data using MySQL and Power BI. The dataset was cleaned, analyzed with SQL queries, and visualized through an interactive Power BI dashboard.

## Tools Used
- MySQL
- MySQL Workbench
- Power BI Desktop

## Dataset
The project uses two datasets:
- matches.csv
- deliveries.csv

## SQL Analysis
The SQL script includes:
- Total Matches
- Total Runs
- Total Balls
- Total Wickets
- Team-wise Match Wins
- Toss Decision Analysis
- Season-wise Match Analysis
- Top Players of the Match

## Power BI Dashboard
The dashboard includes:
- KPI Cards
  - Total Matches
  - Total Runs
  - Total Balls
  - Total Wickets
- Winner Analysis (Bar Chart)
- Toss Decision Distribution (Pie Chart)
- Season Trend (Line Chart)
- Top Players of the Match (Bar Chart)



## Key Insights
- Identified the most successful IPL teams.
- Compared toss decisions across matches.
- Analyzed season-wise trends.
- Identified players with the highest Player of the Match awards.
